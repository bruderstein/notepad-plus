Continued search

After a search inside a file, you can select Search->Find Next or Search->Find Previous to search for this query again. This allows you to quickly perform the same search multiple times without the need for any further input.

Likewise, select Search->Find (Volatile) Next or Search->Find (Volatile) Previous to go find the next/previous word under the carat. Or Search->Go to Next found result or Search->Go to Previous found result to navigate through results shown on the Search result window
